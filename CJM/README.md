몬스터 설계


**몬스터 데이터**
```
int 체력 
int 공격력
int 점수
float 속도
float 총알 발사 주기
enum 타입 ( enum 타입 { 일반, 엘리트, 보스 } )
Vector3 스폰 위치

Item클래스 보유 아이템
```

**몬스터 이동**
```
이동 모드 = 기본 이동 / 타겟 추적 이동

기본 이동
	- 4방향 중 하나 랜덤 이동
	- 정면에 벽이 인식
	- 정면이 벽이면 나머지 3방향 중 랜덤 회전

타겟 추적 이동
	- 타겟 방향으로 방향 고정, 타겟이 사라질 때 까지 이동
	- 시야에 플레이어 들어오면, => 플레이어 추적 이동으로 변경
	- 시야에 플레이어 없으면, 추격 중지 => 기본 이동으로 변경
```

**몬스터 공격**
```
랜덤 주기 발사
	- 기본 이동 중, 랜덤 주기로 발사

타겟 목표 발사
	- 타겟 추적 이동 중, 정해진 주기로 발사
```

**몬스터 피격**
```
총알 충돌체로부터 피격정보 받으면, 주어진 데미지 값만큼 => 체력 -= 데미지
피격 후 일정 시간 동안 무적 (코루틴)
체력이 <= 0 일때 죽음 판정
```

**몬스터 사망**
```
플레이어 스코어 `점수` 만큼 추가
아이템을 들고 있는 몬스터라면 => 아이템 생성
비활성화(오브젝트 풀의 경우) or 파괴
```

**몬스터 매니저**
```
int 최대 몬스터 수
List<Monster> 현재 맵 안에 몬스터 상태

일정 시간마다 몬스터 생성, 리스트에 저장
몬스터 파괴 시, 리스터에서 삭제

리스트에 담긴 몬스터 수가 최대 몬스터 수 이상이면, 몬스터 스폰 멈춤

게임 종료 시 모든 몬스터 초기화
```
